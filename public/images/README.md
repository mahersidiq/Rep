# Product & lifestyle images

Drop your real photos into this folder using the exact filenames below.
They appear on the site automatically — **no code changes needed**. Until a
file exists, a clean flat fallback renders in its place.

PNG with a transparent (or dark) background works best for the cans.

## Product cans — `can-{line}-{flavor}.png`

| File                       | Product            |
| -------------------------- | ------------------ |
| `can-original-citrus.png`  | REP Original Citrus |
| `can-original-mint.png`    | REP Original Mint   |
| `can-original-berry.png`   | REP Original Berry  |
| `can-plus-citrus.png`      | REP Plus Citrus     |
| `can-plus-mint.png`        | REP Plus Mint       |
| `can-plus-berry.png`       | REP Plus Berry      |

## Lifestyle / detail shots

| File               | Used in                                  |
| ------------------ | ---------------------------------------- |
| `lifestyle-1.png`  | "Why REP" — gym/training shot            |
| `pouch.png`        | "Why REP" — close-up of a single pouch   |

Filenames are lowercase, spaces become hyphens. If you add a new flavor later,
follow the same `can-{line}-{flavor}.png` pattern.
